# Movie search website
# React App
Movie API link [http://www.omdbapi.com/](http://www.omdbapi.com/)

To run the project:
1. Clone this repo  `git clone https://github.com/mintesnot96/movie-search-react-website.git`
2. Run `npm install`
3. Run `npm install react`
3. Run `npm start`
